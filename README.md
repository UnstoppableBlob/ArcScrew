# ArcScrew

## Why I made it

Ok, so basically I made this for Science Olympiad in the USA. I am competing in Mission Possible Division B. In the event, there is a task I need to do, which is "raise a marble by 20cm vertically using an archimedes screw". That's why i made this, so i can have a precise archimedes screw which will raise the marble.
(btw i ended up getting 4th place bc of this)

## How to make it 

So you're going to print out the "arc screw.step" file. Then, pick any of "screw1.step", or "screw2.step" to print out, and then it should be good. (This might not work for you, since it might be inaccurate, but I had to change it to work with my specific circumstances, so try your best:)
